Fetch

How to fetch down branches created remotely

In order to update git to know about any remotely created branches via Github a fetch is performed.

  1. To perform a fetch via IntelliJ, go to VCS -> Git -> Fetch

  2. After you have fetched the new branches you can locally checkout the branch to work within that branch
